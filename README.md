## 1. Installation

### 1.a Cloning the repository

```bash
git clone git@github.com:tifo1985/comments.git
```

### 2.c Installation

Run:
```bash
make install
```

## 3. Running the server

When the project is already installed (after a reboot for example), to start it just do a :
```bash
make start
```
